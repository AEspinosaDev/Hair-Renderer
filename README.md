# OpenGL-Hair

- Main branch project runs in Ubuntu >= 24.*.
- Use setup.sh to manage and install all necessary depedencies automatically (Ubuntu only).
- Windows branch has everything already arranged.
- In future versions it will be an platform-agnostic software, but for now, it is what it is.
